```javascript
  country: 'USA',
  currency: 'USD',
  cost: 2.00,
  format: {
    mainPool: { count: 5, range: 69 },
    bonusPool: { count: 1, range: 26, name: 'Powerball' },
    supplementary: null
  },
  draws: ['Monday', 'Wednesday', 'Saturday'],
  jackpotOdds: 292201338,
  anyPrizeOdds: 24.87,
  divisions: [
    { match: '5+PB', prize: 'Jackpot', odds: 292201338 },
    { match: '5+0', prize: 1000000, odds: 11688053 },
    { match: '4+PB', prize: 50000, odds: 913129 },
    // ... etc
  ],
  features: ['Power Play', 'Double Play'],
  taxRate: 0.37 // Federal max
}
```

### Currency Considerations
- Store all values in cents/smallest unit
- Apply exchange rates at display time
- Consider historical rate fluctuations for accuracy

### Odds Calculation Approach
For each lottery type, pre-calculate:
1. Total combinations for jackpot
2. Winning combinations for each tier
3. Expected value per ticket

### Simulation Performance
- Pre-compute prize tier lookup tables
- Use efficient random number generation
- Batch process multiple draws for speed testing

## 10. CONCLUSION

The international lottery landscape offers diverse game mechanics:
- **Best overall odds**: Japan Loto 6 (1:6M) and Spain's El Gordo de Navidad (1:100K)
- **Biggest jackpots**: US Powerball/Mega Millions (potentially $1B+)
- **Most unique mechanics**: Canada's MaxMillions, Japan's tax-free prizes
- **Best value**: Lotto Max (3 plays for C$5), Mega Millions (built-in multiplier)

For the simulator, implementing US and European games first provides the biggest user appeal, while Japanese games offer unique tax-free dynamics worth exploring.
